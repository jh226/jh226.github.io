| 이모지 | 태그        | 의미                            |
|--------|-------------|---------------------------------|
| 🎉     | init        | 초기 커밋, 프로젝트 시작          |
| ✨     | feat        | 새로운 기능 추가                 |
| 🐛     | fix         | 버그 수정                        |
| ♻️     | refactor    | 리팩토링 (기능 변화 없음)         |
| 🔥     | remove      | 불필요한 코드/파일 삭제           |
| 📝     | docs        | 문서 작업 (README 등)             |
| ✅     | test        | 테스트 코드 추가/수정             |
| 🚀     | deploy      | 배포 관련 작업                   |
| 📦     | deps        | 의존성 추가/변경                 |
| ⬆️     | upgrade     | 의존성 업그레이드                 |
| ⬇️     | downgrade   | 의존성 다운그레이드               |
| 🔧     | config      | 설정 파일 수정 (.gitignore 등)    |
| 🗃️     | db          | DB 스키마, 마이그레이션 관련 작업 |
| 🚧     | wip         | 작업 중 (Work In Progress)        |
| 🔀     | merge       | 브랜치 병합                      |
| ⏪     | revert      | 이전 커밋 되돌리기                |
